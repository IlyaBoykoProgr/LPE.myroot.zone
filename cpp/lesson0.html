Для начала,вам нужен компилятор для С++.

Компилятор - программа пробразующая ваш код в готовое приложение.

Если все прошло успешно, откройте Dev-C++. Вот основные комбинации клавиш для работы с Dev-C++:
  • Ctrl+N: создает новый исходный файл
    • Исходным файлом программисты называют файл, в котором находится их код.
  • Ctrl+O: откроет нужный вам исходник(исходный файл) с компьютера
  • Ctrl+S: сохраняет текущий исходный файл
  • Ctrl+W: закрывает текущий исходный файл
  • F11: компилирует и запускает программу
    • Компиляция - процесс создания программы из исходного кода.
Вот еще одно словечко программистов:
IDE (Ай-Ди-Е): программа, в которой пишут исходный код. То есть ваша IDE - Dev-C++.
